## Platform at a Glance

```
AEAOP = AI-powered NOC + SOC + Server Ops + Physical Security
       + Self-Healing + RAG + Autonomous Reporting

Core AI Stack:
  Reasoning:   Qwen3-72B (primary) + DeepSeek-R1-70B (complex analysis)
  Coding:      Qwen2.5-Coder-32B (automation, playbook generation)
  Vision:      YOLO v11 + ByteTrack + Qwen2.5-VL-72B
  Embedding:   nomic-embed-text-v2 (768 dims)
  Serving:     vLLM (production) + Ollama (development)

Data Stack:
  Operational: PostgreSQL 16 + TimescaleDB
  Search/SIEM: Elasticsearch 8.x
  Vector DB:   Qdrant (selected over Milvus/Weaviate/Chroma)
  Cache:       Redis 7 Cluster
  Events:      Apache Kafka
  Objects:     MinIO
  Secrets:     HashiCorp Vault

Application Stack:
  Backend:     FastAPI (Python)
  Frontend:    React + TypeScript + Tailwind CSS
  Agents:      LangGraph + CrewAI
  API Context: MCP (Model Context Protocol)
  Gateway:     Kong + Nginx

Security:
  Identity:    Keycloak OIDC/SSO
  AuthZ:       RBAC + JWT
  MFA:         TOTP / FIDO2
  Network:     Zero Trust + mTLS + Microsegmentation
  Secrets:     HashiCorp Vault
  Audit:       Immutable logs (PostgreSQL + MinIO)
```
